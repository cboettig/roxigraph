# Remove a triple from the store

Remove a triple from the store

## Usage

``` r
rdf_store_remove(store_idx, subject, predicate, object, graph)
```

## Arguments

- store_idx:

  Store index

- subject:

  Subject IRI or blank node

- predicate:

  Predicate IRI

- object:

  Object

- graph:

  Optional graph name IRI
